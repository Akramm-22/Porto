# 🚀 Akram Mujjaman Raton – Portfolio

A modern 3D developer portfolio built with **React**, **Vite**, **Three.js**, **GSAP**, and **Tailwind CSS**. This portfolio showcases my skills, projects, certifications, and experience as a Full Stack Developer.

## 🛠️ Tech Stack

- React
- Vite
- Three.js (React Three Fiber)
- GSAP
- Tailwind CSS
- Framer Motion
- EmailJS

## 🚀 Getting Started

```bash
git clone https://github.com/Akramm-22/Porto.git
cd Porto
npm install
npm run dev
```
